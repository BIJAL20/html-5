Question 1: What are the new tags added in HTML5?

Answer:

Some of the notable new tags added in HTML5 include:

* Section: Represents a thematic grouping of content within a document, such as chapters or subsections. It helps improve the document's structure and semantics.
* Aside: Used for content that is tangentially related to the surrounding text, like sidebars, pull quotes, or advertising.
* Footer: Represents the footer of a section or page, containing metadata, copyright information, and other related content.
* Mark: Highlights or marks a portion of text to make it stand out visually.
* Meter: Represents a scalar measurement within a known range, such as a progress bar or gauge.

Question 4: Canvas and SVG tags

Answer:

canvas Element:

The <canvas> element provides a way to draw graphics, animations, and interactive content using JavaScript.
It works by creating a bitmap-based drawing surface, allowing you to manipulate individual pixels to create various graphical effects.


Svg Element:

The <svg> (Scalable Vector Graphics) element is used to create vector graphics.
Unlike <canvas> SVG is based on XML and represents graphics as a collection of vector shapes, which makes it resolution-independent and suitable for static or interactive graphics.
